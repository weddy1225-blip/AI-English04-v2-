# 🍳 我的創意早餐店 (Breakfast Adventure)

### —— 專為國小四年級設計的「純圖示」英語聽力與邏輯互動遊戲

這是一個基於網頁技術（HTML/CSS/JS）開發的教育互動遊戲。透過模擬早餐店點餐的情境，強迫學生脫離中文字面翻譯，直接將**英語聽力**與**食物圖像**進行連結，建立直覺式的第二語言反應。

## 🎮 遊戲特色
*   **12宮格直覺介面**：全新優化的 6x2 矩陣排列，包含 12 種常用早餐食材，圖示加大且移除英文單字，訓練學生「聽音辨圖」。
*   **純聽力挑戰模式**：預設不顯示文字，學生必須透過聽取 Web Speech API 產生的語音來判斷訂單內容。
*   **精準時間追蹤**：新增「總營業用時」統計，讓學生挑戰反應速度，提升學習動機。
*   **防跑版穩定設計**：採用響應式佈局，確保在平板電腦與電腦螢幕上皆能一目瞭然，無需捲動頁面。

## 📊 營業報告與評估
遊戲結束後將產出詳細的「營業結算報告」，紀錄每一題的作答軌跡。

### 1. 績效標籤
| 標籤 | 代表意義 |
| :--- | :--- |
| **一次過 ✨** | 第一時間聽懂並正確出餐，展現極佳聽力直覺。 |
| **重試 n 次** | 在邏輯判斷或數量計算上出現失誤，經修正後完成。 |
| **看中譯 💡** | 學生尚無法單靠聽力理解，需藉助母語翻譯輔助。 |
| **看句子 📖** | 學生具備閱讀能力，但聽力與詞彙量仍在銜接階段。 |

### 2. 效率指標
*   **總經營用時**：記錄從第一題到結算完成的時間。老師可設定「金牌店長」標準（例如：3分鐘內完成）。

## 🛠️ 技術架構
*   **核心技術**：HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla JS)。
*   **語音系統**：Web Speech API (`window.speechSynthesis`)，語速設定為 0.8 倍速，適合小學四年級聽感。
*   **介面設計**：使用綠、橘、黃暖色調，營造親切的學習氛圍。

## 📝 課綱對應 (國小四年級)
*   **12大核心字彙**：
    *   主食：Egg, Bread, Hamburger, Cake, Cookie, Sandwich.
    *   飲品：Juice, Hot/Cold Milk, Hot/Cold Coffee.
    *   蔬果：Apple.
*   **語音特性**：區分「Hot (♨️)」與「Cold (❄️)」的對比屬性。
*   **學習目標**：能聽辨數量與名詞的組合（如：Two eggs and one juice），並在無文字輔助下做出正確反應。

## 🚀 操作說明
1.  **聽取訂單**：點擊 **🔊 喇叭** 聽取英文要求。
2.  **準備食物**：點擊右側食材櫃的 **圖片按鈕**，食物會進入出餐盤。
3.  **移除錯誤**：若不小心點錯，可點擊餐盤中食材旁的 **✕**。
4.  **求助提示**：若聽不懂，可選擇顯示英文句子或中文翻譯（會記錄在報告中）。
5.  **完成上菜**：點擊 **🔔 確認出餐**。
6.  **查看報告**：完成所有題目後，分析總用時與作答紀錄。
